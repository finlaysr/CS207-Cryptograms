# Job Hunter Example

“As a job hunter, I want to search for a job so that I can find one to
apply for”
- As a job hunter, I want to search for a job based on attributes such as location,
  occupation, and salary range so that I can find one to apply for
- As a job hunter, I want to view the information for each job returned by a search
  so I can determine if I want to apply for it
- As a job hunter, I want to view detailed information about a company who have
  posted a job, so I can determine if I want to work for them
