# User Stories 1 - Tracking progress

## Stakeholder : User

> "As a user, I want to track my progress and others progress on the game, so that I can compare my scores to other users"
* As a user I want my login name saved so that my progress that my personal progress can be saved and accessed.
* As a user I want see my current game progress and other user progresses as a scoreboard afte my game ends, so that I can easily see and compare my score.
